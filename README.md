# 🐢 War Turtle

A top-down shooter game built with Python! Fight off waves of enemies, survive as long as you can, and defeat the boss!!

---

## 🎮 How to Play

- **Arrow keys** — Move your turtle
- **WASD** — Shoot in any direction
- **Click** — Shoot toward mouse *(Easy mode only)*
- **R** — Restart after game over
- **T** — Go back to the title screen

---

## 🕹️ Game Modes

| Mode | Lives | Enemy Speed | Boss At |
|------|-------|-------------|---------|
| 🟢 Easy | 5 | Slow | 50 kills |
| 🟠 Normal | 3 | Medium | 50 kills |
| 🔴 Hard | 1 | Fast | 25 kills |

---

## 👾 Features

- 3 difficulty modes
- Enemies that hunt you down
- Every 5 kills a new enemy spawns (up to 10!)
- A giant BOSS that appears every 50 kills (25 in Hard mode!)
- Boss gets +5 HP every time you defeat it
- Score counter and lives display
- Sound effects
- Title screen
- Game Over screen with restart or return to title

---

## 🚀 How to Run

Make sure you have Python installed, then run:

```
python battle.py
```

No extra installs needed — uses Python's built-in `turtle` module!

> **Note:** Sound effects use `winsound` which only works on Windows.

---

## 🛠️ Built With

- Python 3
- turtle (graphics)
- winsound (sound effects)
- random (enemy spawning)
- threading (sounds without lag)
